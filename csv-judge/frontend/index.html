About the scoring algorithm

Column alignment — headers matched by name, then by content similarity.
Row matching — Hungarian algorithm (optimal) for small datasets; hash + greedy for large.
Cell scoring — exact = 1.0; numerics use relative-difference decay; strings use SequenceMatcher ratio.
Final score = (matched cell scores − missing row penalty) / (ref cells + extra row penalty) × 100
